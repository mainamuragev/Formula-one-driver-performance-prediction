# Formula One Driver Performance Prediction 

This project analyzes Formula One driver performance using machine learning and visualizes results in an interactive Streamlit dashboard.

##  Features
- Data ingestion with FastF1
- Preprocessing and modeling with scikit-learn and TensorFlow
- Interactive visualizations using Seaborn and Matplotlib
- Deployed with Streamlit Cloud

##  Quick Start

### Run Locally
```bash
pip install -r requirements.txt
streamlit run dashboard.py
```
